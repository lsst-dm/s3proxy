### Bug fixes

- Stream S3 object contents instead of buffering entire objects in memory before responding, improving concurrency for workloads such as HiPS tile serving.
- Add optional `Cache-Control` headers for image MIME types via `S3PROXY_CACHE_MAX_AGE`.
- Log object transfer duration and byte counts for S3 GET requests.

### Other changes

- Run uvicorn with uvloop and httptools in the container image.
- Reduce per-request log noise by logging S3 fetch details at debug level only.
